# Face Recognition Demo with DLIB

Simple and fast implementation of a face recognition algorithm using DLIB model library on CPU.

## Requirements

```bash
opencv-python
dlib
face_recognition
numpy
```

# Getting started

## Clone the repository

```bash
git clone 

```

## Install the requirements

```bash
pip install -r requirements.txt
```

or, if using pipenv:

```bash
pipenv install
```

## Run the demo

```bash
python app/main.py
```
